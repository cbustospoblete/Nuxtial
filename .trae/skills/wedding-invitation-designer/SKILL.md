# Diseñador de Invitaciones de Boda

Este skill proporciona guías, estilos y componentes para crear sitios web de invitaciones de matrimonio elegantes y funcionales, inspirados en los mejores referentes del mercado.

## Tipografías Recomendadas

Utiliza estas familias tipográficas (preferiblemente Google Fonts) para lograr el estilo deseado:

### Elegantes y Clásicas (Serif)

- **Playfair Display**: Sofisticada y atemporal. Ideal para títulos principales.
- **Cormorant Garamond**: Fina y de lujo. Perfecta para textos destacados.
- **Cinzel**: Inspiración romana, muy formal. Usar con moderación en encabezados.
- **Bodoni Moda**: Estilo editorial de moda. Alto contraste.
- **Lora**: Romántica y equilibrada. Buena legibilidad en cuerpos de texto.
- **Cardo**: Estilo antiguo y académico. Elegancia clásica.

### Modernas y de Lectura (Sans-Serif)

- **Montserrat**: Moderna y versátil. Títulos y subtítulos limpios.
- **Raleway**: Elegante con trazos únicos. Carácter distintivo.
- **Lato**: Cálida y legible. Excelente para textos largos.
- **Inter**: La mejor para pantallas móviles y UI.
- **Quicksand**: Redondeada y amigable. Toque informal.
- **Tenor Sans**: Diseñada para humanismo y elegancia.

### Caligráficas y de Acento (Script)

- **Great Vibes**: Escritura fluida clásica. Nombres de novios.
- **Pinyon Script**: Lujo extremo y trazos largos. Detalles caligráficos.
- **Dancing Script**: Informal y dinámica.
- **Alex Brush**: Caligrafía corta y muy legible.
- **Amsterdam**: Estilo firma moderna.
- **Parisienne**: Inspiración francesa vintage.

## Referencias de Diseño (Webs)

- **Zola**: Referencia #1 de flujo de usuario (UX). Priorizar la facilidad de uso.
- **The Knot**: Referencia de variedad de plantillas. Ofrecer opciones diversas.
- **Minted**: Referencia de combinaciones de color y tipografía. Armonía visual.
- **Say I do**: Referencia de minimalismo digital. Menos es más.

## Funcionalidades Clave

- **Invitación Digital**: Nombres, fecha, lugar, cuenta atrás.
- **RSVP (Confirmación)**: Formulario simple y claro.
- **Historia**: Timeline o narrativa de la pareja.
- **Galería**: Fotos de la pareja (grid masonry o carrusel).
- **Regalos / Mesa de Regalos**: Enlaces a listas o información bancaria.
- **Información Útil**: Mapas, hoteles, dress code.

## Tech Stack & Implementación

- **Nuxt 4**: Aprovechar SSR para SEO y rendimiento.
- **Tailwind CSS 4**: Utilizar clases utilitarias para maquetación rápida y consistente.
- **Componentes Vue**: Crear componentes reutilizables (ej. `WeddingButton`, `SectionTitle`, `CoupleName`).
- **Responsive Design**: Mobile-first es CRÍTICO, ya que la mayoría de invitados verán la invitación en el móvil.

## Paletas de Color Sugeridas

1. **Clásico Romántico**: Blanco roto, Rosa palo, Dorado, Gris carbón.
2. **Botánico Natural**: Verde salvia, Blanco puro, Madera, Verde oscuro.
3. **Elegancia Nocturna**: Azul marino, Plata, Blanco, Negro suave.
4. **Minimalista Moderno**: Blanco, Negro, Gris suave, Acento terracota.

## Instrucciones de Uso

Cuando se pida diseñar una sección o página:

1.  Identificar el estilo (Clásico, Moderno, Rústico, etc.).
2.  Seleccionar la combinación tipográfica adecuada (ej. Playfair + Lato).
3.  Proponer el layout y los componentes necesarios.
4.  Implementar con Tailwind CSS asegurando la responsividad.
